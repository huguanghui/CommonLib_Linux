使用dnsmasq搭建DNS服务器

搭建环境

  • Virtualbox + Ubuntu
  • dnsmasq

安装

1 # apt方式安装
2 $ sudo apt-get install dnsmasq
3 # 服务开启
4 $ service dnsmasq start

配置

配置文件位置/etc/dnsmasq.conf

DNS服务器添加域名和IP的映射表

在配置文件中搜索address=,找到相关的配置位置,按照下面格式,添加想要映射的关系表.

1 {c++}
2 # 添加domain到IP的映射
3 address=/usdev.zviewcloud.com/192.168.6.95

注意: 如果想要映射关系立即生效,请重启服务

使用

1 $ service dnsmasq start # 启动服务
2 $ service dnsmasq restart # 重启服务
3 $ service dnsmasq stop # 停止服务
4 $ service dnsmasq force-reload # 强行加载
5 $ service dnsmasq status # 查看运行状态和log
6 $ chkconfig dnsmasq on # dnsmasq设为开机自启

开放了防火墙端口有时无法解析域名

1 $ service iptables stop # 关闭防火墙
2 $ chkconfig iptables off #关闭防火墙的开机自启

P2P相关的地址表

测试zwhosts时在/etc/dnsmasq.conf添加如下的域名映射

# Add domains which you want to force to an IP address here.
# The example below send any host in double-click.net to a local
# web-server.
address=/double-click.net/127.0.0.1
address=/usdev.zviewcloud.com/47.88.33.190
address=/szdev.zviewcloud.com/119.23.131.217
address=/szdev.aftx.net/120.76.126.62
address=/erdev.zviewcloud.com/47.91.88.40
address=/bjdev.aftx.net/123.57.156.223
address=/hzdev.aftx.net/121.43.96.85
