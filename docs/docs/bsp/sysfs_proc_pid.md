------
## status

> 进程状态信息

```
Name：进程名称
State：进程状态（例如running、sleeping等）
PPid：父进程的进程ID
Pgrp：进程组ID
Uid：用户ID
Gid：组ID
VmSize：进程虚拟内存的大小
VmRSS：进程物理内存的大小
Threads：线程数
FDSize：进程打开文件的数量
```
